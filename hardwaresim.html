HOW TO USE
■ PLACING COMPONENTS
Drag any component from left panel onto the canvas. Or double-click to auto-place.

■ WIRING
Press W or click Wire tool. Click a component pin, then click another pin. Green = signal HIGH.

■ SIMULATION
Click ▶ RUN to start auto-clock. Use STEP for manual advance. Toggle buttons/switches by clicking them.

■ KEYBOARD
S=Select, W=Wire, D=Delete, F=Fit, Ctrl+Z=Undo, Delete=Remove
Alt+Drag=Pan, Scroll=Zoom

■ SERIAL MONITOR
View real-time output from running code. println() calls appear here.

■ EXAMPLES
Click ⚡ EXAMPLES header button to load preset circuits.
